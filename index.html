How to Use & Install

  1. Go to https://github.com/Akirane/XIVHotbar
  2. Click the green Code button, then click Download ZIP
  3. Extract the XIVHotbar - master folder inside the .zip file into your Windower > addons folder
  4. Rename the XIVHotbar - master folder to just XIVHotbar
  5. Go to Windower > scripts and edit the init.txt file
  6. Add lua l xivhotbar to the end of the file, save and close
  7. Use this website to build your bars with your desired actions for your class
  8. Click the Export .lua button once you're done
  9. Two files will be downloaded: a General file and your in-game class file
  10. Copy both of them to Windower > addons > XIVHotbar > data > YourCharacterName (create this folder if it doesn't exist)
  11. Launch the game!
